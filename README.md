# Employee Management System

## Overview
A simple Employee Management System that allows users to:
- Add new employee
- Edit employee details
- View employee 
- Delete employee

This project demonstrates basic CRUD operations and RESTful APIs.

## Tech Stack
- **Backend:** Spring Boot
- **Frontend:** Angular
- **Database:** MySQL with JPA
  ## Features
- App employee, View , Update, Delete employee
